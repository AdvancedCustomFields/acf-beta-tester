Get access to the latest versions of Advanced Custom Fields

== Description ==

This plugin allows you to update Advanced Custom Fields to any tagged release on the [WordPress repository](https://wordpress.org/plugins/advanced-custom-fields/) including Early Acces, Beta, and previous version!

== Installation ==

From your WordPress dashboard

1. **Visit** Plugins > Add New
2. **Search** for "Advanced Custom Fields Beta Tester"
3. **Activate** Advanced Custom Fields Beta Tester from your Plugins page
4. **Visit** Plugins and find the "Advanced Custom Fields" plugin (note the new options available)
5. **Select** the version you would like to download
6. **Update** the Advanced Custom Fields plugin to the selected version
